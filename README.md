# Quiz Paper Generator Android App

Android WebView app for:

https://quiz-paper-generator.emergent.host/

## Build online from GitHub

1. Extract the ZIP.
2. Upload all extracted files and folders to the root of a GitHub repository.
3. Open the repository's **Actions** tab.
4. Open **Build Android APK**.
5. Press **Run workflow**.
6. When the run finishes, open it and download the
   **Quiz-Paper-Generator-APK** artifact.
7. Extract the downloaded artifact ZIP to get `app-debug.apk`.

The app needs internet because it loads the live website.
Any future update made at the same website URL will appear automatically.
